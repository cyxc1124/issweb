# 陈阳小草Iss服务器列表

使用 [flask] 构建后台。  
使用 [python-a2s] 查询服务器信息查询。

## 本地运行

运行 `python3 app.py` 用浏览器打开 [http://localhost:8090](http://localhost:8090)

## 部署

关于flask部署 [Deployment Options]。 这里放了一个我正在使用的gunicorn文件 `gunicorn.conf`。
